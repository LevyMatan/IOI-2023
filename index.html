Naive Algorithm

The naive algorithm works by iterating over each element of A and finding the smallest sum of an element of A and an element of B that has the current element of A as the first element. This is done by iterating over each element of B and computing the sum of the current element of A and the current element of B. If the sum is smaller than the current minimum sum, it is updated as the new minimum sum. Once all elements of B have been checked, the minimum sum is added to the output vector C.

Code

std::vector<int> smallest_sums(int N, std::vector<int> A, std::vector<int> B) {
  std::vector<int> C(N);

  for (int i = 0; i < N; i++) {
    int min_sum = A[i] + B[0];
    for (int j = 1; j < N; j++) {
      int sum = A[i] + B[j];
      if (sum < min_sum) {
        min_sum = sum;
      }
    }
    C[i] = min_sum;
  }

  return C;
}
